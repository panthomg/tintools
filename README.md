# NoteForge - Beautiful Writing Experience

A modern, feature-rich writing application designed for creating books, notes, and capturing thoughts with elegance and functionality.

![NoteForge Logo](icon-192x192.png)

### Core Writing Features
- **Rich Text Editor**: Powered by Quill.js with comprehensive formatting options
- **Real-time Word & Character Count**: Track your writing progress
- **Auto-save**: Never lose your work with automatic saving every 30 seconds
- **Multiple Export Formats**: Export to TXT, HTML, Markdown, and JSON
- **Document Management**: Organize and manage multiple documents

- **Beautiful, Modern UI**: Clean and aesthetic design with attention to detail

- **Dropbox Sync**: Secure cloud storage and synchronization
- **Cross-device Access**: Access your documents from any device

- **Keyboard Shortcuts**: Efficient shortcuts for power users

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for initial setup and cloud sync

### Installation

#### Option 1: Direct Use
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start writing immediately!

#### Option 2: Local Development Server
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Or serve built files
npm run serve
```

#### Option 3: Install as PWA
1. Open the app in your browser
2. Look for the "Install" prompt or use browser menu
3. Install NoteForge as a native app

### Dropbox Integration Setup
1. Create a Dropbox app at [Dropbox App Console](https://www.dropbox.com/developers/apps)
2. Get your App Key
3. Open Settings in NoteForge
4. Enter your Dropbox App Key
5. Click "Connect" to authenticate
